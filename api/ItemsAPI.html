class ItemsAPI

Gives access to all the things you can do with the items API of Echo.

Constructor

Constructor Parameters Returns
ItemsAPI( options )

Parameters:

  • options <object> Options, need to specify:
    • options.consumerKey : Echo API Key
    • options.consumerSecret : Echo API Secret
    • options.feed : Feed to communicate with
    • options.authHandler : optional, in ['oauth','basic']. Default 'oauth'
